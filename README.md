# Details
Made in Unreal Engine mostly with blueprints, log system implemented in C++.

# Controls
| Key  | Action |
| ------------- | ------------- |
| WASD | Walk Around |
| F | Try to extinguish the candle |
| Spacebar | Scan area in front of you |
| Arrow Up/Down | Switch options in the main menu |
